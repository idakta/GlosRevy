3. Purpose, goals and audience

The purpose of the website is to give Gløshaugen Revy- og teaterlag a website wher it is easy to find information about the revue and theathergroupe they recently started. GRT's website will contain all possible information you need to know, such as genral information, all the sub-groups that it is possible to join, the board of GRT and where to find them. Our goal is to make a website to spread information and news about GRT, and in this way make it easy for the audince to find answers to things they wonder about GRT.

The websites audience will be everyone who is interested in the revue, whether you want to join in yourself or just watch a performance.
The main audience will probably be students. But the website is also relevant for older visitors, for example parents or gradnparants. Therfore we can imagine audinces of all ages stopping by the website, so it is important to have a neat and clear website where all target groups and age groups manage to navigate to the different pages.


4. Navigation Structure

The navigation structure of GRT's website can be described as a clique. By this we mean it will be posible to access all of the pages included in the website, from the homepage via the menu. All the pages will contain the menu, which gives you accsess to all pages regardless of which page you are on. The menu will also contain a drop-down menu for the varios subgroups. This will be visible when the "Blimedlem"-button is hovered upon. The subgroups will all be at the same page, so that you can scroll down and read about the different groups. You can also automatically get down to a spesific subgroup by using the drop-down menu and click on the subgroup you want to read about.

We choose a clique structure because we think this is the best way to make a neat and clear website where it is easy to navigate. Since the website audience potentially can be all age groups it is important for us to make a user-frindly website, as we think is with a clique structure. This will give the user freedom to easyly accsess the different pages by clicking around on the website. As shown in the figure under whatever page you are on give you accsess to who ever you want to visit next, and it's easy to go back to the homepage.


5. Page Layout and Appearance

We have an ongoing theme which is "classy" and "high-end". The font we use is called "Korolev", and the standard font size is 30. The client also provided 8 different colors which they wish that we use. The color codes are: #F2D398 #F2C84B #D9A036 #BF7C2A #8C501B #262626 #000000 #FFFFFF.

The navigation bar is positioned at a standard place for many similar websites, so that it is easy to find. It is in the same color scheme as the rest of the website, but different from the elements nearest so that it is easy to spot and catches the user’s eye quickly. In addition, it is used as a separator between the banner and the rest of the page. When (clicking? Hovering?) over certain elements in the bar, a list of subtopics will appear, making it easier for the user to navigate the site.

The banner is classy and simple. It contains the client’s logo and does not overshadow the rest of the content on the pages.

The footer is styled with the same color as the navbar, and the text is styled in the same way. It will include links to Facebook, Instagram and Tik Tok, that are portrayed with their standard icons in white, making them stand out. It will also include contact information.

The website should be easy to navigate and have all information visible and quickly found. In addition, as the client is in need of a serious website, it should look classy and serious, which is why we chose the simple, elegant design.
